# PBL_PPUPIN

PPUPIN(퍼핀)은
개인정보수집이용동의서가
최신 개인정보보호법과
가이드라인을 준수하는지
확인할 수 있게 해주어
사용자의 개인정보보호를 돕는
서비스 입니다.

### PPUPIN의 기능(서비스/특징/등등)
    ▪️ 동의서의 점수와 위험 단계: 동의서의 완성도와 위험 단어를 고려하여 점수를 산출해 보여줍니다.
    ▪️ 동의서의 완성도: 개인정보보호법을 참고해 선정한 다섯 가지 항목의 만족 여부를 OX로 보여줍니다.
    ▪️ 동의서의 위험 단어: 동의서에 존재하는 위험 단어(개인정보영향평가 안내서에 제시된 개인정보 등급표 참고)와 위험 이유를 보여줍니다.
    ▪️ 자세히 보기:
	    - 완성도와 위험 단어에 대한 설명: 완성도와 위험 단어가 무엇인지에 대해 설명합니다.
	    - 완성도와 위험 단어 세부 점수: 동의서의 완성도와 위험 단어 각각의 점수를 보여줍니다.
	    - 동의서의 완성도: 메인화면에서와 마찬가지로 다섯 가지 항목의 만족 여부를 OX로 보여줍니다.
	    - 동의서 내의 위험 단어와 해당 부분: 동의서에 존재하는 위험 단어와 해당 단어가 있는 부분을 보여줍니다.
	    - 전체 위험 단어 목록: PPUPIN에서 검사하는 전체 위험 단어의 목록과 그 이유를 표로 보여줍니다.
